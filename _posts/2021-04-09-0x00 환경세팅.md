# 0x00 환경세팅

msvc 컴파일러 : Visual Studio 2017/2019 

gcc 컴파일러 : 거의 모든 채점 서버 사용

`#include <bits/stdc++.h>`

bits 폴더 안의 stdc++.h에 미리 수많은 헤더 들을 include 해두어 코드에서 쓰는 헤더들을 하나하나 include하는 번거로움을 덜어준다. 

gcc 컴파일러에서 사용 가능, msvc에서도 설정해주면 사용 가능

1. `/usr/local/include` 경로 이동
2. `bits` 폴더 생성
3. `bits` 폴더 안에 `stdc++.h` 파일 생성
4. 아래 내용 복사 붙여넣기 후 저장
